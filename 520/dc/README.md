# waimai

超级外卖 Super Cms是专业的外卖订餐系统，使用它，不用写代码，只要会打字，就可以管理网站。
前台设计采用采用html5与css3设计，兼容IE6+、Firefox、Chrome、Safari、Opera等主流浏览器.
并可以在微博，微信中完美显示。后台功能模块化设计，用户操作方便。 易于上手，即安即用。
适合餐馆，酒店，外卖平台，糕点店，海鲜店
